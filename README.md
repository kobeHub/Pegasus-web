# pegasus-web

The web client of [Pegasus](github.com/kobeHub/Pegasus) based on vuejs, which is under developing.

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
